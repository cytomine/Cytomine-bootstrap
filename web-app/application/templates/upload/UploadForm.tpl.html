Important notes !

  • The maximum file size (per file) for uploads is 100 GB
  • Only image files (JP(E)G, PNG, BMP, TIF, TIFF, JP2K, SVS, SVS J2K, SCN, NDPI, VMS, MRXS) are allowed
  • VMS and MRXS must be zipped (the VMS/MRXS file and their nested files. One zip per image.
  • A Zip file containing multiple image is allowed (except for VMS and MRXS)
  • You can drag & drop files from your desktop on the 'Add files...' button with Google Chrome, Mozilla Firefox and Apple Safari.
  • You are also able to link images with project manually after upload is done.
